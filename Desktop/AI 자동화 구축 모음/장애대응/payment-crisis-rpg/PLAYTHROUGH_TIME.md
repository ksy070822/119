# 클릭 진행 시 예상 소요 시간

처음부터 끝까지 **클릭/스킵만으로** 진행할 때의 예상 시간입니다.  
(대사 읽지 않고 바로 진행하는 “최소 시간”과, 약간씩 읽으며 진행하는 “실제 플레이”를 구분했습니다.)

---

## 1. 전체 요약

| 구간 | 최소 (클릭만) | 실제 플레이 (대략) |
|------|----------------|---------------------|
| **타이틀** | ~0분 | ~0분 |
| **인트로** | ~0분 10초 | ~0분 30초 ~ 1분 |
| **본편 (S1~S5)** | ~1분 30초 | ~4분 ~ 6분 |
| **보스전** | ~0분 20초 | ~0분 40초 ~ 1분 |
| **엔딩** | ~0분 10초 | ~0분 30초 ~ 1분 |
| **합계** | **약 2분 ~ 2분 30초** | **약 6분 ~ 9분** |

- **최소**: 스킵/클릭만으로 기다리지 않고 진행할 때.
- **실제 플레이**: 대사·선택지·연출을 조금만 읽고 진행할 때.

---

## 2. 장면별 상세

### 2-1. 타이틀 (TitleScene)

- **동작**: “스토리 참여하기” 1회 클릭 → 인트로로 이동.
- **최소**: 1클릭, 거의 0초.
- **실제**: 0분 (바로 클릭).

---

### 2-2. 인트로 (IntroScene)

- **구성**: 6장면 (평화 마을 → 기운 → 결제대란 → 영웅 소집 → 캐릭터 선택 → 컨트롤센터 집결).
- **클릭으로 진행**:
  - 장면 0→1→2→3: 클릭(또는 Space) 시 다음 장면으로 전환 (전환 페이드 약 0.6초).
  - 장면 4: 캐릭터 선택 1회 클릭.
  - 장면 5: **자동 5초** 후 게임씬으로 이동 (클릭으로 건너뛰기 불가).
- **스킵 버튼**: “스킵 >>” 클릭 시 곧바로 캐릭터 선택(장면 4)으로 이동.

| 구간 | 최소 (클릭만) | 비고 |
|------|----------------|------|
| 장면 0~3 | 4클릭 + 전환 4×0.6초 ≈ 2.4초 | 클릭 시 즉시 다음 장면 |
| 캐릭터 선택 | 1클릭 | |
| 장면 5 (집결) | **고정 5초** | 자동 진행 |
| **인트로 합계** | **약 7~10초** | 스킵 시 캐릭터 선택까지 1클릭, 이후 5초 |

---

### 2-3. 본편 (GameScene) — S1 ~ S5

- **진행**: 스테이지마다 NPC 4명 대화 → (관장자 안내 1회) → 관장자 대화 → 결정(선택지) → 다음 스테이지 전환.
- **진행 단위**: 대사/선택/응답마다 1회 “진행(클릭 또는 Space)” 필요.  
  (타이핑은 50ms/글자, 클릭 시 즉시 스킵 가능.)

**스테이지별 대략적인 “진행(클릭)” 수**

| 스테이지 | NPC 대화 | 관장자 안내 | 관장자 대화 | 결정(대사+선택+응답) | 스테이지 합계 |
|----------|----------|-------------|-------------|----------------------|----------------|
| S1 장애 인지 | 8 | 1 | 2 | 3 | **14** |
| S2 초기 공지 | 8 | 1 | 2 | 3 | **14** |
| S3 심화 공지 | 8 | 1 | 1 | 3 | **13** |
| S4 복구 판단 | 8 | 1 | 2 | 3 | **14** |
| S5 후속 조치 | 5 | — | — | 3 | **8** |
| **합계** | **37** | **4** | **7** | **15** | **63** |

- **스테이지 전환**: S1→2, 2→3, 3→4, 4→5 네 번.  
  전환당 페이드 인 0.5초 + 표시 0.5초 + 페이드 아웃 0.7초 ≈ **1.7초 × 4 ≈ 6.8초**.
- 결정 전 짧은 딜레이 0.5초 1회.
- “아이템 전부 획득” 연출: 스킵 가능, 클릭 시 약 0.3~0.4초 수준.

| 구간 | 최소 (클릭만) | 비고 |
|------|----------------|------|
| 63회 진행 | 63×0.15~0.2초 ≈ 9~13초 | 클릭 반응만 가정 |
| 스테이지 전환 4회 | 약 7초 | 고정 연출 |
| 기타 (딜레이·연출) | 약 1~2초 | |
| **본편 합계** | **약 1분 30초 ~ 2분** | 빠르게 클릭 시 |
| **실제 플레이** | **약 4~6분** | 선택지/대사 읽을 때 |

---

### 2-4. 보스전 (BossScene)

- **구성**: 9페이즈 (몬스터 등장 → 보스 등장 → 보스 대사 → 나레이터 → 영웅 등장 → 아이템 합체 → 외침/필살기 → 보스 격파 → 승리).
- **진행**: 페이즈마다 클릭(또는 Space)으로 다음 페이즈.  
  페이즈별 **최소 표시 시간**이 코드에 정의되어 있음.

| 페이즈 | 내용 | 최소 지속(코드 기준) |
|--------|------|----------------------|
| 0 | 몬스터 미니 인카운터 | 2.0초 |
| 1 | 보스 등장 | 2.0초 |
| 2 | 보스 대사 | 1.5초 |
| 3 | 나레이터 | 1.2초 |
| 4 | 영웅들 등장 | 2.0초 |
| 5 | 아이템 합체 | 1.5초 |
| 6 | 영웅 외침/필살기 | 1.5초 |
| 7 | 보스 격파 | 1.5초 |
| 8 | 승리 | 2.0초 |
| **합계** | | **약 15.2초** |

- **최소**: 9클릭 + 최소 지속 합 15.2초 → **약 17~20초**.
- **실제**: **약 40초 ~ 1분**.

---

### 2-5. 엔딩 (EndingScene)

- **진행**:
  - “마을의 평화가 찾아왔습니다” 문구: **1.8초** 대기.
  - 주민 말풍선 2개: 각 1.2초 + 마지막 0.8초.
  - 그 다음 하단 바에 나레이션 한 줄씩 표시 (줄당 약 2.5초 표시 + 0.4초 전환).
- **스킵**: “스킵” 클릭 시 위 연출 생략 후 곧바로 **등급·타이틀 버튼** 표시.
- **마지막**: “수고하셨습니다.” 클릭 → (필살기 동영상 있으면 재생, 없으면 메시지 후) **1.5~1.8초** 뒤 타이틀로 복귀.

| 구간 | 최소 (클릭만) | 비고 |
|------|----------------|------|
| 스킵 사용 시 | 1.8초 + 1클릭(스킵) → 등급 표시 | |
| “수고하셨습니다” 클릭 | 1클릭 | |
| 동영상 없을 때 | 1.5~1.8초 후 타이틀 | |
| **엔딩 합계 (스킵)** | **약 5~10초** | |
| **엔딩 (스킵 없이)** | **약 25~35초** | 나레이션 줄 수에 따라 |

---

## 3. 총 소요 시간 (정리)

- **클릭/스킵만으로 최대한 빠르게**  
  **약 2분 ~ 2분 30초**  
  (타이틀 0초 + 인트로 ~10초 + 본편 ~1분 30초 + 보스 ~20초 + 엔딩 ~10초).

- **대사·선택지 조금 읽으며 플레이**  
  **약 6분 ~ 9분**  
  (인트로 30초~1분, 본편 4~6분, 보스 40초~1분, 엔딩 30초~1분).

- **모든 대사·연출을 보며 플레이**  
  **약 15~25분**  
  (인트로 자동 8+8+8+18초 구간, 본편 타이핑·선택 고려, 보스·엔딩 연출까지).

위 수치는 코드(대사 수, 전환 시간, 페이즈 최소 시간)와 클릭 1회당 0.15~0.2초 가정으로 산정한 예상치입니다.
